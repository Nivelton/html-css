5 < 2 --> true
7 > 4 --> false
8 >= 8 --> true
9 <= 7 --> false
5 == 5 --> true
4 != 4 --> false
5 === "5" --> false
5 !== "5" --> true

Ordem de soma


1º Os peradores aritiméticos
2º Os peradores relacionais
3º Os peradores lógicos

Precedência

1º (), **, *, /, %, +, - 2º >, <, >=, (quem aparecer primeiro) 3º !, &&, ||


exemplos

idade >= 15 && idade <= 17 // a idade está entre 15 e 17? estado == "RJ" || estado == "SP" // o estado é RJ ou SP? salário > 1500 && sexo != "M" // o salário é acima de R$1,500,00 e não é homem?
